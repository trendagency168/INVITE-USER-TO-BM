> Professional Facebook Business Manager User Inviter Tool with multi-threading support

## 🌟 Features

- ✅ **Multi-thread Processing**: Invite multiple users simultaneously
- ✅ **User Support**: Invite using Facebook User ID or Email address
- ✅ **Smart Duplicate Check**: Automatically skip already invited users
- ✅ **Role Management**: Assign Admin or Employee roles
- ✅ **Real-time Logging**: Color-coded progress updates
- ✅ **Copy Results**: One-click copy of success/failed lists
- ✅ **Check Invites**: Verify if users are already invited
- ✅ **List Invited Users**: View all invited users in the BM
- ✅ **Premium UI**: Modern design with gold TREND AGENCY branding
- ✅ **Draggable Interface**: Move tool anywhere on screen

## 📋 Prerequisites

- Facebook Business Manager account with Admin access
- Modern web browser (Chrome, Firefox, Edge, Safari)
- JavaScript enabled

## 🚀 Quick Start

### Method 1: Bookmarklet (Recommended)

1. Copy the code from `bookmarklet.txt`
2. Create a new bookmark in your browser
3. Paste the code as the URL
4. Click the bookmark on Facebook Business Manager page

### Method 2: Browser Console

1. Navigate to your Facebook Business Manager
2. Open Developer Tools (`F12` or `Ctrl+Shift+I`)
3. Go to Console tab
4. Paste the script from `script.js` and press Enter
